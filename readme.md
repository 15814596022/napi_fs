### 生成对应平台的.node
```
// target "x86_64-apple-darwin", "x86_64-unknown-linux-gnu", "i686-pc-windows-msvc", "x86_64-pc-windows-msvc", "aarch64-apple-darwin" ...
yarn build --target aarch64-apple-darwin
```

node test
